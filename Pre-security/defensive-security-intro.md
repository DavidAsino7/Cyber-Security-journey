# Defensive Security Intro

**Path:** Pre Security  
**Platform:** TryHackMe  
**Date completed:** July 14, 2026

## What I learned
- Defensive security focuses on protecting systems, monitoring for threats, 
  and responding to attacks — the counterpart to offensive security
- Introduced to core defensive roles/concepts (e.g. SOC analysts, threat 
  detection, incident response)

## Key takeaway
Where offensive security finds weaknesses before attackers do, defensive 
security is about detecting and responding when something goes wrong in 
real time.
