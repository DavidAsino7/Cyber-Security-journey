# Offensive Security Intro

**Path:** Pre Security  
**Platform:** TryHackMe  
**Date completed:** July 14, 2026

## What I learned
- The difference between offensive security (attacking to find weaknesses) 
  and defensive security (monitoring/protecting systems)
- Introduced to `dirb`, a directory/file brute-forcing tool used to discover 
  hidden pages or files on a web server by testing a wordlist of common names
- Ran my first commands on a live Linux machine: `whoami`, `pwd`

## Key takeaway
Offensive security is about thinking like an attacker before a real attacker 
does — finding gaps proactively rather than reacting after a breach.
